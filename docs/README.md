**5 Real Projects We've Shipped:**

   - Investment: $10K (80 hours)

   - Problem: App had 47 TypeScript errors, service worker issues
   - Solution: Fixed TypeScript strict mode, PWA compliance, performance optimization
   - Result: 0 TypeScript errors, 3.2s → 1.1s load time, 98 Lighthouse score
   - Investment: $1.6K (13 hours, emergency rate)

3. **Gratog** - Project management tool
   - Problem: Team used spreadsheets, hard to track progress
   - Solution: Kanban board, real-time collaboration, dashboard, notifications
   - Result: 90% team adoption, 95% on-time tasks, 80% fewer status meetings
   - Investment: $15K (120 hours)

### Our Philosophy
- **Clear pricing** - $125/hour (or $65 partner), no hidden markup
- **Focus on UX** - Users matter more than features
- **Simple MVPs** - Ship fast, iterate with feedback
- **Transparent process** - Weekly demos, no surprises
- **Real results** - Measure what works, optimize

### Pricing Tiers
```
Starter:       $2,500/month  (20 hrs)   or  $1,300 partner
Professional:  $7,500/month  (60 hrs)   or  $3,900 partner
Enterprise:    $20,000+/month (160+ hrs) or $10,400+ partner
```

### How We Estimate
1. **Project type baseline** (20-100 hours)
2. **Design complexity** (+15-40 hours)
3. **Database/backend** (+10-30 hours)
4. **Integrations** (+5-30 hours per integration)
5. **Special requirements** (compliance, performance, SEO, training)
6. **Team adjustments** (non-tech +10 hours, strong dev no change)

1. **Fill inquiry form** (10 minutes)
