### 3. User Submits Inquiry
```
User fills out 6-section form
‚Üì

‚Üì

‚Üì
API calculates estimate based on:

  - Mobile platforms (+hours each)
  - Timeline & team level (adjustments)
‚Üì
Returns pricing tier & cost breakdown
‚Üì
User sees estimate with "We'll contact you"
```

```
User visits site, pages are cached

‚Üì
Service worker serves cached pages
‚Üì
Static content works perfectly
‚Üì
APIs gracefully fail (user can retry when online)
```
